#
# Community definitions file
#
#
# All lines beginning with '#' or empty are ingored. Other lines should be
# formatted as
#
#	<name><tab..><value>
#
# The only exception is AS start tag in form
#
#	AS<num>
#
#
# Global variables
#	All global variables are upper case. Variable name length should be
#	equal with digit count. All letters should be the same.
#
#	A=1		one digit variable
#	XXX=010		three digit variable
#
#
# Local variables
#	All local variables are lower case. Variable name length should be
#	equal with digit count. All letters should be the same. Variables
#	are usable only within the current AS list and will be deleted after
#	next AS num start tag.
#
#	n=0		one digit variable
#	cccc=1234	four digit variable
#
#
# Community lists for any AS should begin with start tag.
#
#	AS555
#	AS12345
#
#
# Public community documents can be referred with special tag "URL"
#
#	URL	http://noc.domain.net/communities.txt
#
#
# Communities can be in normal form with description
#
#	123:111		test community
#	65500:9999	something else..
#
# or can include variable names in them. In this case all strings in form of
# $<variable name> will be replaced with given variable description
#
#	123:300xx	Customer route from $xx
#	1234:3xxyK	Prepend $xx to $y $K times
#
